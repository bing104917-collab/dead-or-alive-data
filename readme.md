## 1.下载与安装
**电脑端：**
首先需要下载node.js,推荐版本**20.19.4**以后。
在命令行用 :

```powershell
npm install
npx expo start --clear
```
安装。

![效果图](images/example.png)
可以打开浏览器输入相关的http://localhost:port 运行
也可以使用手机扫描二维码运行

**手机端：**
直接下载apk文件即可运行。
